# Agent Rules for MyCities Project

**STRICT RULES - DO NOT TAKE LIBERTIES**

1. **Implement exactly what is specified** - Do not add features, labels, or UI elements that were not explicitly requested.

2. **Do not add labels or text** that was not specified (e.g., "Liters", "Kiloliters").

3. **Follow exact specifications:**
   - Water reading RED font region: accepts up to **6 digits**
   - Water reading BLACK font region: accepts up to **4 digits**

4. **Clear button behavior:** Pressing Clear on the keypad completely clears the active section to display **0**.

5. **Back button:** Required at the bottom of reading screens.

6. **Screen separation:** Water and Electricity Enter Reading are **separate screens**.

7. **No assumptions** - If unclear, ask before implementing.
